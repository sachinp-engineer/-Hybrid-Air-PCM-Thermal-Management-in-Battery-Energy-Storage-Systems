# 🔋 Real-Time Embedded Control Platform for Hybrid Air-PCM Thermal Management in Battery Energy Storage Systems

## 📖 Overview

This project presents a **Real-Time Embedded Control Platform** for managing the temperature of lithium-ion battery energy storage systems using a **Hybrid Air–PCM (Phase Change Material) Thermal Management System**.

The system continuously monitors battery temperature, voltage, and current using an ESP32 microcontroller. Based on real-time sensor data, the controller automatically operates the cooling system to maintain safe battery operating temperatures, improving battery performance, reliability, and lifespan.

## 🚀 Working Principle

1. The ESP32 continuously reads battery temperature, voltage, and current.
2. Sensor data is processed in real time.
3. When the battery temperature exceeds the predefined threshold, the controller activates the cooling system.
4. Hybrid Air–PCM cooling reduces battery temperature and improves thermal uniformity.
5. The system continues monitoring until safe operating conditions are restored.
